OpenNG is developed **in the open**. Source for the API, docs, scrapers, and infrastructure tooling lives in one monorepo.

## Repository

**[github.com/stephcrown/openng](https://github.com/stephcrown/openng)** — clone, open issues, and submit pull requests against `main` following existing patterns in each package.

## Participate

- **[Contributing](/contributing)** — data fixes, documentation, and code expectations
- **[Roadmap](/roadmap)** — what is planned next

License terms are defined in the **root of the repository** when a `LICENSE` file is present; check the latest tree on GitHub before you redistribute or fork.

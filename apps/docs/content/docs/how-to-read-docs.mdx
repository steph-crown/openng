## Page layout

Each page has a **title**, optional **description**, and body copy. The right column lists headings on this page so you can jump while reading.

## Code blocks

Shell samples use `curl`. JSON fragments are illustrative—always verify against live responses.

## Cross-links

Internal links use root-relative paths (`/rate-limits`). They work on `docs.openng.dev` with `baseUrl` set to `/`.

## Conventions

Tables summarize endpoints and codes. Callouts highlight caveats without breaking the main flow.

OpenNG uses **passwordless email** for the web app and **bearer tokens** for programmatic access.

## Web (magic link)

1. Enter your email on the main app.
2. Click the link we send—single-use, short-lived.
3. A session cookie is set for dashboard routes (`HttpOnly`, `Secure`).

## API keys

- Prefix: `ong_live_` followed by a random segment.
- Send as `Authorization: Bearer ong_live_...`.
- Keys are hashed at rest; you only see the full value once when created.

## Anonymous use

Omit `Authorization` to call public `GET` endpoints; you share anonymous rate limits by IP.

## Do I need an API key?

No for basic use. Keys raise rate limits and unlock future paid capabilities.

## Is there GraphQL?

Not currently—only REST under `/v1`.

## Where is the OpenAPI spec?

Generated from the API and published with the docs as the project matures; until then use `/meta` and each resource’s `/meta` route.

## How do I report bad data?

See [Contributing](/contributing). Include sources and the affected resource.

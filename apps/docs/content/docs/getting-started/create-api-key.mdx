## Web dashboard

Sign in on the main OpenNG app with your email. Open **Dashboard → API keys** and create a key. The full token is shown **once**—store it in a password manager or secret store.

## Format

Keys use the prefix `ong_live_` followed by a high-entropy segment. Send the entire string as:

```
Authorization: Bearer ong_live_…
```

## Rotation

Revoke the current key, then create a new one. Old keys stop working immediately once revoked.

icon: Tags
---

**Resource** — A dataset served at `/v1/{name}` with list, `meta`, and detail routes generated from one ResourceFactory config.

**Envelope** — JSON shape with `success`, optional `data` or `error`, and `meta` on list responses.

**Staging** — Import tables where rows are validated before promotion to `public`.

**Batch** — Identifier tying imported rows together for validation and migration jobs.

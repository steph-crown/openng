## Base URL

| Environment | API base |
| --- | --- |
| Production | `https://api.openng.dev` |
| Local | `http://localhost:3000` (when running the API stack) |

All versioned resources live under `/v1`.

## Make a request

```bash
curl -sS "https://api.openng.dev/v1/holidays?limit=3"
```

You should receive a JSON **envelope** with `success: true`, a `data` array, and `meta` (totals, pagination, resource metadata).
